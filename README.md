### Installation

#### Recommended Installation Method

- Download the File

```curl -o StudioInstaller.sh https://raw.githubusercontent.com/pinedamg/Android-Studio-Installer-Script/master/StudioInstaller.sh```

- Make it as Executable

```chmod +x StudioInstaller.sh```

- Run it

```./StudioInstaller.sh```

> Once the Installation is Complete, You'll see a Message **Installing Finished**
> Open your Launcher and Check for Android Studio
